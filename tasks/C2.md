You are assisting with project management analysis. Follow the output format exactly.

## Task

A software product team must prioritize six backlog items using RICE scoring. Use the formula RICE = (Reach x Impact x Confidence) / Effort. Use the numbers exactly as given, do not adjust them.

| Item | Description | Reach (users/quarter) | Impact | Confidence | Effort (person months) |
|---|---|---|---|---|---|
| B1 | Mobile app dark mode | 12000 | 1 | 0.8 | 6 |
| B2 | One click checkout | 3000 | 2 | 1.0 | 2 |
| B3 | AI powered search | 6000 | 2 | 0.5 | 4 |
| B4 | Bulk export to CSV | 900 | 3 | 0.8 | 1 |
| B5 | Faster page loads | 15000 | 0.5 | 1.0 | 3 |
| B6 | In app referral program | 2500 | 2 | 0.7 | 5 |

Compute the RICE score for each item, rank them from highest to lowest score, and give a one sentence justification for the top ranked item.

## Output format

Respond with a single JSON object and nothing else. No prose before or after, no code fences.

{
  "scores": {"B1": 0, "B2": 0, "B3": 0, "B4": 0, "B5": 0, "B6": 0},
  "ranking": ["B?", "B?", "B?", "B?", "B?", "B?"],
  "top_justification": "one sentence"
}
