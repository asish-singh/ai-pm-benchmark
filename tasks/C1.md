You are assisting with project management analysis. Follow the output format exactly.

## Task

A marketing team must prioritize five campaign initiatives using RICE scoring. Use the formula RICE = (Reach x Impact x Confidence) / Effort. Use the numbers exactly as given, do not adjust them.

| Initiative | Description | Reach (people/quarter) | Impact | Confidence | Effort (person months) |
|---|---|---|---|---|---|
| F1 | Email nurture sequence | 4000 | 2 | 0.8 | 4 |
| F2 | Webinar series | 1000 | 3 | 1.0 | 2 |
| F3 | SEO landing page refresh | 8000 | 0.5 | 0.5 | 1 |
| F4 | Partner co marketing pilot | 500 | 3 | 0.8 | 3 |
| F5 | Retargeting ad campaign | 2000 | 1 | 0.9 | 1 |

Compute the RICE score for each initiative, rank them from highest to lowest score, and give a one sentence justification for the top ranked item.

## Output format

Respond with a single JSON object and nothing else. No prose before or after, no code fences.

{
  "scores": {"F1": 0, "F2": 0, "F3": 0, "F4": 0, "F5": 0},
  "ranking": ["F?", "F?", "F?", "F?", "F?"],
  "top_justification": "one sentence"
}
